# 타임오프클럽 — 개발 기능 정리

## 📱 멤버 화면

### 로그인 페이지 (`/login`)
- 🌿 "3월, 44명의 멤버와 함께하고 있습니다" 배지
- 매주 수요일 저녁 8시 · 일요일 오후 3시 안내
- "일주일에 한 번 운동하듯, 정기적인 OFF를 루틴으로" 문구
- 🛒 멤버십 가입하기 버튼 (네이버 스마트스토어 연결)
- 하단 카카오톡(well__moment) + 인스타그램 문의
- 라벨 "본인 확인 코드"로 변경 (서비스다운 표현)
- 모바일 가독성 최적화 (줄바꿈, 폰트 사이즈)

### 캘린더 (`/calendar`)
- 날짜별 예약 인원수 표시 ("3명")
- 휴무 날짜 "휴무" 표시 (관리자가 닫은 날짜)
- 월별 멤버십 제한 — 비활성월로 넘기면 결제 안내 모달 + 스마트스토어 링크
- 헤더 "타임오프클럽" / "OO님, 환영합니다" 줄바꿈 방지

### 스팟 선택 (예약)
- 스팟별 예약 인원 + 💬스몰토크/🧘사색 비율 표시
- 닫힌 스팟은 "휴무"로 표시, 선택 불가
- 스팟별 안내 메시지 표시 (관리자 설정, 예: "수요일은 8시에 맞춰 오픈")
- 소인원 넛지 — "소규모로 더 깊은 대화" 추천
- 마감 임박(80%) 🔥 / 소규모(3명 이하) ✨ 뱃지
- 불편 멤버 회피 — 과거 신고한 세션 참여자가 같은 스팟에 있으면 경고
- **사색 모드 선택 시 활동 선택** (일기 쓰기, 책 읽기, 편지 쓰기, 드로잉, 생각 정리, 그냥 쉬기) + 준비 가이드

### 내 예약 (`/calendar` 오른쪽)
- 모드 뱃지 크게 (💬 스몰토크 / 🧘 사색), "모드" 텍스트 제거
- 당일 예약 → "🌿 오늘의 타임오프 준비하기" 버튼

### 세션 준비 (`/session-ready`)
- 날짜 + 스팟 + 💬스몰토크/🧘사색 인원수
- 오늘의 참가자 목록 (익명)
- **스몰토크 모드** → 🗨️ 오늘의 대화 카드 (랜덤 질문)
- **사색 모드** → 🧘 오늘의 사색 카드 (읽기 명상, 자유 글쓰기, 소리 관찰, 감사 리스트 등 10가지)
- 📵 스마트폰 보관 안내 + ⏰ 정시 시작 안내
- "📵 스마트폰을 내려놓을 준비가 되었습니다" 버튼 → 마무리 화면

### 내 기록 (`/my-history`)
- 참석 세션 수, 방문 스팟, 쉼의 시간
- 모드 뱃지 크게 (💬/🧘)
- 🚨 불편 신고 버튼 + 모달

### 피드백 (`/feedback`) — 탭 3개
1. **💬 피드백** — 세션별 만족도 + 서비스 개선 + 불편사항(관리자만 확인) + 오늘의 한마디(익명)
2. **🧘 사색 회고** — 사색 참석 세션 선택 → 느낀 점, 깨달음, 다음에 하고 싶은 것 기록 → 나만의 사색 일지
3. **📮 요청** — 운영자에게 바라는 점 (카테고리별) + 운영자 답변 확인

### 스몰토크 가이드 (`/guide`)
- 스몰토크 팁 + 대화 카드

## 🏠 스팟 운영자 화면 (`/admin/spot`)

- 체크인 날짜 자동 설정 (가장 가까운 수/일)
- ✅ 참가자 체크 (출석/노쇼 버튼)
- 예약 현황 (멤버 뒷4자리 + 모드 표시)
- 변경/취소 로그
- 📢 웰모먼트 공지 확인
- **📮 웰모먼트에 요청** — 건의사항 작성 + 웰모먼트 답변 확인

---

## 👑 웰모먼트 슈퍼관리자 (`/admin`)

### 기존 기능
- 예약 관리, 로그, 멤버 관리 (활성월 설정)
- 체크인 관리, 체험권, 💬 피드백 + 🚨 불편 신고

### 새로 추가된 기능
- **📮 요청** — 멤버 + 스팟 운영자 요청 통합 관리, 👤멤버/🏠운영자 뱃지 구분, 답변 작성
- **📅 캘린더 관리**
  - 🚫 날짜 닫기 (전체 또는 특정 스팟)
  - 👥 날짜별 스팟 인원 제한 설정
  - ℹ️ 스팟별 안내 메시지 설정
- 📢 공지 관리
- 🚨 신고 관리

## 🗄️ DB 테이블 (신규)

| 테이블 | 용도 |
|--------|------|
| `operator_requests` | 멤버/운영자 → 웰모먼트 요청 |
| `closed_dates` | 닫힌 날짜/스팟 |
| `spot_notices` | 스팟별 안내 메시지 |
| `reflection_logs` | 사색 회고 일지 |
