## 콘텐츠 카테고리 (피드)

### 1. 브랜드 소개 / 철학 (30%)
- 타임오프클럽이 뭔지, 왜 시작했는지
- 디지털 디톡스의 가치
- "오늘, 폰을 내려놓아 볼까요?"

### 2. 세션 안내 / 일정 (30%)
- 매주 수/일 세션 안내
- 스팟별 분위기 소개
- 참여 방법 안내

### 3. 감성 / 무드 (20%)
- 세션 중 분위기 (대화, 사색, 공간)
- 참여자 후기 (익명)
- 오프라인의 소중함

### 4. 참여 유도 (20%)
- 체험권 안내
- Q&A
- 이벤트/챌린지

## 릴스 아이디어

1. "2시간 동안 폰 없이 보낸 사람들의 표정 변화"
2. "카페에서 폰 대신 ___를 했다" (사색, 대화, 글쓰기)
3. "스크린타임 vs 리얼타임" 비교
4. 세션 공간 분위기 스케치 (BGM + 공간 영상)

---

## 스토리 활용

- 매주 세션 전: 오늘의 세션 안내 (시간, 장소)
- 세션 중: 현장 분위기 실시간 공유
- 세션 후: 하이라이트 + 한줄 후기
- 일상: 투표, Q&A, 디톡스 팁

---

## 게시 빈도 (추천)

- 피드: 주 2~3회
- 릴스: 주 1회
- 스토리: 매일 또는 세션일 집중

## 초기 테스트 콘텐츠 3개

1. **브랜드 소개** — "타임오프클럽이란?" 카드뉴스형 이미지
2. **세션 안내** — 매주 수/일 세션 시간·장소 안내
3. **감성 무드** — "2시간, 폰 없이" 분위기 이미지
