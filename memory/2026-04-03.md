### 해결한 문제들
1. **4/26일 스티키플로어**: 예약자 0명인데 고아 대화 주제가 보이던 문제
2. **4/12일 스티키플로어+다시점**: 대화 열기 버튼이 안 보이던 문제

### 구현한 개선사항

#### 1️⃣ 자동 고아 주제 정리 시스템
- `cleanupOrphanedTopics()` 함수 추가
- 예약 생성/삭제 시 자동으로 스몰토크 예약자 0명인 스팟의 주제 삭제
- 관리자용 `/api/conversation-topics/cleanup` API 추가

#### 2️⃣ 대화 주제 → 스몰토크 자동 설정
- 대화 주제 설정 시 자동으로 스몰토크 모드 선택
- 사색 모드 선택 시 대화 주제 해제 확인 알림

#### 3️⃣ 하이브리드 활성화 조건
- 기존: 4명 이상 스팟 있을 때만 활성화  
- 신규: **4명 이상 스팟 OR 전체 예약자 6명 이상**일 때 활성화
- 빈 날에도 대화 열기 가능하면서 스팟 밸런싱 효과 유지

#### 4️⃣ 따뜻한 UI 메시지
- 기존: "💬 주제명"
- 개선: "💬 '주제명' 주제로 함께 이야기 나누고 싶어요"

#### 5️⃣ 사색 예약자 제외 로직
- 사색 예약자는 대화 주제 조건 계산에서 제외
- 사색 1명 + 스몰토크 0명 = 여전히 대화 열기 가능
