# 타임오프클럽 브랜드 확장 로드맵
## — 디지털 디톡스를 축으로 어디까지 확장할 수 있을까

> 작성일: 2026-04-22
> 전제: 디지털 디톡스 = 브랜드 핵심 / 스몰토크 + 사색 = 현재 Mode 2개
> 목표: 핵심 정체성 유지하면서 수평·수직으로 확장 가능성 탐색

---

## 🧭 확장 원칙 — 지킬 것 / 바꿀 것

### 🔒 절대 바꾸지 말 것 (브랜드 DNA)
- **📵 Digital OFF** — 폰이 없는 경험
- **🙊 Resume OFF** — 이름·직업 묻지 않음
- **🎈 Outcome OFF** — 결과 강요하지 않음
- **소규모·차분함** — 친밀함이 가능한 규모
- **절제된 미감** — 자연 톤, 여백, 차분함

### 🔄 바꿔도 되는 것 (확장의 축)
1. **Mode** — 2시간 안에 무엇을 하는지 (현재: 스몰토크/사색)
2. **Time** — 언제 모이는지 (현재: 수 20시 / 일 15시)
3. **Space** — 어디서 모이는지 (현재: 약수·망원·압구정)
4. **Depth** — 얼마나 오래 (현재: 2시간)
5. **Audience** — 누구를 위한 것인지 (현재: 도시 직장인)
6. **Product Form** — 어떤 형태로 경험하는지 (현재: 오프라인 모임)

---

## 🔀 Axis 1. Mode 확장 — 2시간 안의 '무엇'

스몰토크·사색 외에 브랜드 결에 맞는 Mode를 추가.

| 신규 Mode | 설명 | 브랜드 적합도 |
|---|---|---|
| 📖 **읽기 모드** | 각자 책 가져와 조용히 묵독. 원하면 마지막 15분 나눔. | ⭐⭐⭐ 매우 적합 |
| ✍️ **쓰기 모드** | 저널링·편지·에세이. 손글씨 중심. | ⭐⭐⭐ 매우 적합 |
| 🎨 **만들기 모드** | 필사·컬러링·드로잉·자수 같은 정적 손활동. | ⭐⭐⭐ 매우 적합 |
| 🧘 **움직임 모드** | 가벼운 스트레칭·명상·요가. 말 없이. | ⭐⭐ 적합 (공간 필요) |
| 🎧 **듣기 모드** | 큐레이션된 음악·사운드힐링 감상. | ⭐⭐ 적합 |
| 🍵 **맛보기 모드** | 차 시음·간단한 식문화 경험. 말없이 감각에 집중. | ⭐⭐⭐ 매우 적합 |

**추천 다음 단계**: 읽기 / 쓰기 / 만들기 중 1~2개를 5월에 파일럿 도입.
**마케팅 파생**: "오늘의 모드를 고르세요 — 스몰토크 · 사색 · 읽기" 확장된 선택지로 콘텐츠 리뉴얼.

## 🕐 Axis 2. Time 확장 — '언제'

현재는 평일 저녁/일요일 오후. 다른 시간대 페르소나를 커버.

### 🌅 아침 타임오프 (가능성 ⭐⭐⭐)
- **시간**: 평일 오전 7-8시
- **활동**: 짧은 명상 + 저널링 + 간단한 차
- **타겟**: 출근 전 뇌를 자극 없이 시작하고 싶은 직장인
- **장점**: 기존 멤버 중복 안 되는 완전히 새로운 페르소나

### 🌑 심야 타임오프 (가능성 ⭐⭐)
- **시간**: 평일 밤 22-24시
- **활동**: 사색 + 저널 + 수면 가이드
- **타겟**: 불면·번아웃 직장인
- **장점**: "잠이 안 오는 사람들"이라는 강력한 페르소나

### 🌞 주말 긴 타임오프 (가능성 ⭐⭐⭐)
- **시간**: 토요일 오후 4시간
- **활동**: 모드 3~4개 로테이션 (사색 → 쓰기 → 스몰토크 → 차 마시기)
- **가격**: 단독 상품 (예: 80,000원) 또는 멤버십 상위 등급
- **장점**: 프리미엄 라인 구축

### ⏱ 점심 타임오프 (가능성 ⭐)
- **시간**: 평일 12-13시 (30분)
- **활동**: 짧은 명상·사색
- **주의**: 30분은 너무 짧아 "진짜 OFF" 효과 약함. 우선순위 낮음.

## 🏞 Axis 3. Space 확장 — '어디서'

현재는 도심 카페형 공간. 공간이 바뀌면 경험도 바뀌고 타겟도 바뀜.

### 기본형 (도심 카페·공간)
현재 약수·망원·압구정. 5월 신규 2곳 추가. 이게 코어.

### 🌳 자연 타임오프 (분기별 특별 이벤트)
- **장소**: 한강변, 북한산 자락 벤치, 양재천
- **활동**: 산책 사색 + 자연 속 스몰토크
- **특징**: 시즌 한정 (봄·가을). FOMO 유도 가능.

### 📚 서점 콜라보 타임오프
- **장소**: 독립 서점 (최인아책방, 책바, 땡스북스 등)
- **활동**: 서점 큐레이션 책 중 한 권 묵독 후 스몰토크
- **장점**: 서점은 자연스러운 디톡스 공간. 서점 측도 이벤트 수요 있음. 비용 낮음.

### 🖼 갤러리 타임오프
- **장소**: 한적한 소규모 갤러리
- **활동**: 작품 앞에서 사색 + 감상 스몰토크
- **장점**: "제3의 공간" 컨셉 강화. 고급 페르소나 유입.

### 🛕 사찰 타임오프 (심화 상품)
- **장소**: 서울 근교 사찰 (진관사, 봉은사 등)
- **활동**: 반나절 또는 하루 템플스테이형
- **장점**: 이미 디지털 디톡스 효과 검증된 공간. 프리미엄화 가능.

### 🏨 호텔/스테이 콜라보
- **장소**: 소규모 부티크 호텔
- **활동**: 1박 2일 디지털 디톡스 스테이
- **장점**: 호텔 객실 판매 + 타임오프 수익 배분. B2B 모델.

## ⏳ Axis 4. Depth/Duration 확장 — 얼마나 깊게

### 현재: 2시간 (진입 상품)

### 🌿 반나절 리트릿 (4시간)
- 기존 2시간 확장판. 모드 여러 개 체험.
- 가격: 50,000~60,000원
- 포지셔닝: "2시간은 부족했던 분들을 위한"

### 🌿🌿 하루 리트릿 (1Day, 8시간)
- 월 1~2회 이벤트
- 자연 공간 활용 (예: 양평 게스트하우스 낮 시간)
- 가격: 120,000~150,000원

### 🌿🌿🌿 1박 2일 리트릿 (심화)
- 교외 게스트하우스 + 호텔
- 월 1회 한정, 10명 내외
- 가격: 250,000~350,000원
- **의미**: 타임오프클럽의 "정점 경험"

### 📅 4주 챌린지 프로그램 (온·오프라인 혼합)
- 매주 1회 오프라인 모임 + 온라인 주간 리추얼 카드
- 5월 멤버십의 업그레이드 버전이 될 수 있음
- 가격: 80,000~100,000원 (현재 멤버십보다 프리미엄)

## 👥 Axis 5. Audience 확장 — 누구를 위해

현재는 20-30대 도시 직장인. 번아웃 자각자. 이걸 다변화.

### 직군별 특화
- **의료진 타임오프** — 간호사·의사 번아웃 직군
- **교사 타임오프** — 방학 시즌 특화
- **창작자 타임오프** — 작가·디자이너
- **부모 타임오프** — 육아 번아웃 (아이 맡기고 2시간 쉼)

### 연령별 특화
- **50+ 타임오프** — 은퇴·전환기 세대
- **청년 타임오프** — 20대 전반 (조금 더 저렴하게)

### 상황별 특화
- **새내기 타임오프** — 입학·입사 1년 차
- **전환기 타임오프** — 이직·이별 같은 삶의 변화 중

> ⚠️ 주의: 너무 많이 분화하면 브랜드가 파편화. 1~2개만 파일럿 추천.

## 📦 Axis 6. Product Form 확장 — 어떤 형태로

### 🎁 굿즈 / 물리적 제품 (브랜드 적합도 ⭐⭐⭐)

브랜드 감성과 잘 맞는 제품.

- **폰 보관 박스** (집에서 쓸 수 있는 작은 버전)
- **리추얼 저널** — 타임오프 프롬프트가 들어간 일기장
- **리추얼 엽서 세트** — 자신에게 쓰는 편지 시리즈
- **차 블렌딩 키트** — 스팟별 시그니처 차
- **음악 플레이리스트 카드** — QR 없이 설명만 있는 아날로그 카드

> ✅ 굿즈는 **모임 없는 날에도 브랜드를 경험**하게 해주는 장치.

### 📖 책 / 뉴스레터 (브랜드 적합도 ⭐⭐)
- 타임오프클럽 에세이집 (운영자가 저자)
- 주간 뉴스레터 (디지털 디톡스 인사이트)
- ⚠️ 콘텐츠 과잉이 될 수 있음. 절제 필요.

### 📱 디지털 제품 (브랜드 적합도 ⭐)
- 앱·디지털 도구는 **브랜드 역설**. 웬만하면 하지 말 것.
- 유일한 예외: **소리 없는 결제·예약 시스템** (카톡 봇 X, 간단한 웹 폼만)

### 🏢 B2B 프로그램 (브랜드 적합도 ⭐⭐⭐)
- **기업 팀빌딩** — 네트워킹 없이 진짜 쉬는 워크샵
- **HR 번아웃 프로그램** — 분기별 사내 타임오프
- **학교·대학 프로그램** — 청소년·대학생 디지털 웰빙
- **의료기관 프로그램** — 의료진 번아웃 케어

> ✅ B2B는 **수익 안정성 + 브랜드 확산** 동시 효과. 우선순위 높음.

## 🚦 절대 하지 말아야 할 것 (브랜드 역설)

이런 건 아무리 유혹적이어도 **브랜드 정체성을 파괴**해요.

- ❌ **네트워킹 이벤트** — Resume OFF 훼손
- ❌ **앱·챗봇** — Digital OFF 역설
- ❌ **100명+ 대규모 행사** — 친밀함 파괴
- ❌ **결과물 필수 워크샵** (자격증, 수료증) — Outcome OFF 훼손
- ❌ **자기계발·성과 지향 프로그램** — 브랜드 톤 완전 붕괴
- ❌ **인플루언서 콜라보 (시끄러운 스타일)** — 차분함 훼손
- ❌ **과한 인스타 공유 유도** — "진짜 OFF"와 모순

## 📅 단계별 로드맵 제안

### 🌱 Phase 1. 지금 ~ 6월 — 코어 강화
*"하는 건 같은데 훨씬 풍부해진다"*

- Mode 1~2개 추가 (읽기 / 쓰기 / 만들기 중 택)
- 스몰토크 3단계 카드 도입
- 스탬프·저널 누적 시스템
- 5월 신규 스팟 2곳 정착

### 🌿 Phase 2. 7~9월 — 수평 확장
*"타임오프클럽이 여기에도 있어요"*

- **아침 타임오프** 파일럿 (주 1회, 1개 스팟)
- **서점 콜라보 스팟** 1곳 오픈
- **반나절 리트릿** 월 1회 파일럿
- 콜라보 B2B 1개사 (소규모 스타트업 대상)

### 🌳 Phase 3. 10~12월 — 깊이 확장
*"타임오프를 더 깊게 경험할 수 있다"*

- **1박 2일 리트릿** 정식 상품화
- **4주 챌린지 프로그램** 출시
- 자연 타임오프 (가을 시즌)
- B2B 2~3개사 확장

### 🌲 Phase 4. 내년 — 제품·커뮤니티
*"모임 밖에서도 타임오프 경험"*

- **굿즈 라인** 출시 (저널·엽서·박스)
- 연간 시즌제 운영 (분기별 특별 시즌)
- 특정 오디언스 특화 (1개 직군 타겟)

## 💰 수익 구조 다변화 관점

**현재**: 월 멤버십 40,000원 × N명 = 단일 수익원

**확장 시**:
| 상품 | 가격대 | 역할 |
|---|---|---|
| 월 멤버십 | 40,000원 | 진입 상품 (현재) |
| 반나절 리트릿 | 50~60,000원 | 체험 상품 |
| 4주 챌린지 | 80~100,000원 | 심화 상품 |
| 1일 리트릿 | 120~150,000원 | 이벤트 상품 |
| 1박 2일 리트릿 | 250~350,000원 | 프리미엄 상품 |
| B2B 워크샵 | 300~500만원/회 | 안정 수익 |
| 굿즈 | 15~60,000원 | 보조 수익 + 브랜드 노출 |

→ **포인트**: 진입은 쉽게, 깊이는 깊게. 멤버가 "더 갈수록" 선택할 수 있는 상위 상품이 있어야 해요.

## 🎯 브랜드 확장이 마케팅에 주는 파급효과

확장은 그 자체로 **새로운 타겟에 도달**할 기회예요.

| 확장 | 새롭게 열리는 타겟 |
|---|---|
| 아침 타임오프 | 아침형 직장인, 러닝 크루 |
| 읽기 모드 | 독서 모임 참여자 |
| 서점 콜라보 | 독립 서점 커뮤니티 |
| 반나절·1박 리트릿 | 요가·명상·웰니스 타겟 |
| B2B 프로그램 | HR·조직문화 관심층 |
| 굿즈 | 라이프스타일 쇼핑 타겟 |

→ 기존 타겟(도시 직장인) 외에 **요가/웰니스 씬·HR 씬·독서 씬**으로 브랜드 확장 가능.

## ⭐ 가장 먼저 실험할 만한 3가지 (6월 안에)

위 모든 것 중 **가장 적은 리소스로 가장 큰 임팩트**가 나올 3가지:

### 1. Mode 추가 — 읽기 또는 쓰기 (Phase 1)
- 소요: 2주
- 리스크: 낮음 (기존 스팟 그대로)
- 임팩트: "2시간 콘텐츠가 얇다" 문제 즉시 해소 + 마케팅 소재

### 2. 아침 타임오프 파일럿 (Phase 2 앞당김)
- 소요: 1개월
- 리스크: 중간 (새 시간대 수요 미지수)
- 임팩트: 완전 새로운 페르소나 확보 + 기존 멤버도 추가 구매 가능

### 3. B2B 파일럿 1개사 (Phase 2 앞당김)
- 소요: 1개월 (리드 구하고 제안하기)

## 🔁 확장 전에 확인할 질문

확장 결정 전에 스스로 답해보면 좋을 질문들:

1. **이 확장이 '디지털 디톡스'의 진짜 의미를 더 잘 전달하나, 아니면 흐리나?**
2. **현재 멤버가 이 확장 얘기 들으면 "맞아, 타임오프답다" 할까 "어, 그건 다른 브랜드 같은데" 할까?**
3. **운영 복잡도가 얼마나 늘어나나? 운영자 1명이 감당 가능한가?**
4. **이것 없이도 브랜드가 성장할 수 있나? 아니면 반드시 필요한가?**
